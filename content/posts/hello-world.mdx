# Hello World

Welcome to my blog! This is the first post written as a local **MDX** file.

## Why MDX?

MDX lets you write Markdown with embedded React components. That means I can:

- Write normal Markdown content
- Use **code blocks** with syntax highlighting
- Embed custom components like videos
